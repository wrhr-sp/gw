# 통합설정 2차 비밀번호 미리보기 흐름 안내

## 한 줄 요약

지금 미리보기에서는 설정 관련 화면에 들어가기 전에 2차 비밀번호 4자리 PIN을 먼저 확인하고, 통합설정은 항상 `기본 설정` 탭부터 열리며, 2차 비밀번호 변경 버튼은 관리자설정 우상단이 아니라 `내정보 설정` 안에 있습니다.

## 이 문서가 설명하는 범위

- 통합설정 진입 전 2차 비밀번호 확인 흐름
- `내정보 설정` 안에서 2차 비밀번호를 설정/변경하는 위치
- 4칸 네모형 PIN UI 기준
- 이번 미리보기가 어디까지 되고, 어디까지는 아직 아닌지

## 먼저 알아둘 점

1. 이번 기능은 아직 미리보기입니다.
2. 2차 비밀번호는 현재 서버나 DB에 저장되지 않고, 화면 상태로만 동작합니다.
3. 그래서 새로고침하거나 세션이 바뀌면 다시 처음 상태처럼 보일 수 있습니다.
4. 해시 저장, 실패 횟수 제한, 감사 로그, 운영자 초기화는 아직 이번 범위가 아닙니다.

## 사용자에게 보이는 실제 흐름

### 1) 설정 관련 화면에 들어가면 먼저 2차 비밀번호를 확인합니다

- 통합설정
- 내정보 설정
- 관리자설정처럼 설정 성격의 진입점

위 화면으로 들어가려고 하면 먼저 2차 비밀번호 확인 카드가 뜹니다.

안내 문구 기준:
- 제목: `2차 비밀번호`
- 설명: `설정 관련 기능에 들어가기 전에 4자리 PIN을 확인합니다.`

## 2) 입력 UI는 모두 같은 4칸 네모형 PIN 방식입니다

모든 2차 비밀번호 입력은 같은 모양을 씁니다.

- `2차 비밀번호`
- 네모 4칸
- 숫자 4자리만 입력 가능
- 입력한 숫자는 화면에 그대로 보이지 않음
- 입력된 칸과 비어 있는 칸이 구분됨
- 오류 문구는 4칸 PIN 아래에 표시됨

즉, 한 줄짜리 일반 비밀번호 입력창처럼 보이지 않고, 네 칸짜리 PIN 입력처럼 보이게 맞춘 상태입니다.

## 3) 아직 2차 비밀번호가 없으면 바로 설정으로 이어집니다

아직 등록된 2차 비밀번호가 없으면 게이트 카드 안에 아래 흐름이 보입니다.

- 안내 문구: `아직 2차 비밀번호가 없습니다.`
- 보조 문구: `이번 미리보기에서는 서버 저장 없이 화면 상태로만 2차 비밀번호를 설정합니다.`
- 버튼: `2차 비밀번호 설정하기`

이 버튼을 누르면 2차 비밀번호 설정 팝업으로 이동합니다.

## 4) 통합설정은 인증 후에도 항상 `기본 설정` 탭부터 열립니다

2차 비밀번호 확인을 마치고 통합설정으로 들어가면,

- 첫 화면은 항상 `기본 설정`
- 관리자 계정이어도 처음부터 `관리자설정` 탭으로 열지 않음

즉, `통합설정 = 기본 설정부터 시작` 이 현재 기준입니다.

## 5) 관리자설정 우상단에는 더 이상 2차 비밀번호 변경 버튼이 없습니다

이번 최신 기준에서는 예전처럼 관리자설정 우상단에서 2차 비밀번호를 바꾸지 않습니다.

확인 포인트:
- 관리자설정 우상단 `2차 비밀번호 변경` 버튼 없음
- 관리자설정은 통합설정 안의 한 탭으로만 봄
- 비밀번호 변경 위치는 `내정보 설정` 으로 이동

## 6) 2차 비밀번호를 바꾸려면 `내정보 설정`으로 갑니다

변경 위치는 아래 순서입니다.

1. `내정보 설정` 열기
2. 보안 성격 섹션에서 `2차 비밀번호 변경하기` 버튼 확인
3. 버튼을 누르면 2차 비밀번호 팝업 열기
4. 이미 등록된 상태라면 현재 PIN 확인 후 새 PIN 입력

현재 문구 기준:
- 버튼명: `2차 비밀번호 변경하기`

## 7) 설정 팝업과 변경 팝업의 차이

### 처음 설정할 때

- 현재 PIN 입력칸 없음
- 새 PIN 4자리 입력
- 확인 PIN 4자리 입력
- 저장 성공 시 설정 완료 문구 표시

### 이미 있는 PIN을 바꿀 때

- 현재 PIN 4자리 입력
- 새 PIN 4자리 입력
- 확인 PIN 4자리 입력
- 현재 PIN이 맞아야 변경 가능

## 8) 지금 단계에서 운영 기능처럼 보면 안 되는 부분

아래 항목은 아직 이번 미리보기 범위가 아닙니다.

- 서버 저장
- API 검증
- DB 반영
- 해시 저장
- 실패 횟수 제한
- 잠금 정책
- 감사 로그
- 운영자 초기화
- production 데이터 반영

즉, 지금 문서는 `화면에서 사용자가 어떻게 따라가면 되는지`를 설명하는 용도이고, 운영 보안 완료 문서가 아닙니다.

## 직접 확인할 때 빠른 순서

1. 설정 관련 화면 진입
2. 2차 비밀번호 4칸 PIN 게이트가 먼저 뜨는지 확인
3. PIN 미등록 상태면 `2차 비밀번호 설정하기` 버튼이 보이는지 확인
4. 등록 후 통합설정이 `기본 설정` 탭부터 열리는지 확인
5. 관리자설정 우상단에 변경 버튼이 남아 있지 않은지 확인
6. `내정보 설정` 안에 `2차 비밀번호 변경하기` 버튼이 있는지 확인
7. 설정/변경/게이트가 모두 같은 4칸 PIN UI를 쓰는지 확인
