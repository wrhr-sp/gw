# We’reHere 그룹웨어 기능 아이콘 7차 초안

## 기준
- 사용자가 선택한 공개 SVG 후보를 실제로 가져와 반영했습니다.
- 보고는 정정대로 Lucide `notebook-pen.svg` 메모/작성형 보고 아이콘입니다.
- 지점 아이콘은 제외했습니다. 지점은 우측 상단 `지점관리포털` 버튼으로 분리합니다.
- 사용자가 말하지 않은 아이콘은 기존 초안을 유지했습니다.

## 선택 반영
- 메신저: Phosphor `chat-circle.svg`
- 근태: Phosphor `clock.svg`
- 게시판: Heroicons `newspaper.svg`
- 보고: Lucide `notebook-pen.svg`
- 전자결재: Tabler Icons `clipboard-check.svg`
- 급여: Lucide `circle-dollar-sign.svg`
- 세무: Tabler Icons `receipt-tax.svg`
- 노무: Lucide `hard-hat.svg`
- 파일: Heroicons `document.svg`
- 설정: Tabler Icons `settings.svg`
- 경영대시보드: Lucide `chart-no-axes-combined.svg`

## 파일
- `svg/*.svg`: 개별 SVG
- `png-selected/*.png`: 실제 SVG 렌더링 PNG
- `groupware-icons.sprite.svg`: sprite 초안
- `groupware-feature-icons-preview.png`: 미리보기
- `ICON_SOURCES.md`: 출처/라이선스 정리

## 주의
이 단계는 앱 연결 완료가 아니라 디자인 에셋 7차 초안입니다.


## 비율 조율
- 선택한 공개 아이콘은 유지했습니다.
- Phosphor 256 viewBox 아이콘은 24 viewBox 기준 wrapper로 정규화했습니다.
- 미리보기는 모든 아이콘을 동일한 22px 렌더링 박스 안에서 확인합니다.
